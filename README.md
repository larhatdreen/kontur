# Установка зависимостей
```bash
npm install
```

# Запуск проекта
```bash
npm run dev
```

# Описание проекта
```js
В данном проекте были некоторые моменты, которые я бы хотел замоноложить для полного понимания картины.

В Figma указан шрифт Poppins. Из-за отличий в шрифтах и особенностей жирности Poppins, визуал может отличаться от макета. Жирность не всегда соответствует ожидаемой, можно провести тестирование.

UI содержит спорные моменты: либо это проверка разработчиков, либо некачественный дизайн. Дизайн table отсутствует — это риск, так как восприятие у разных разработчиков разное. Промежуточный дизайн обязателен для контроля качества.
```
